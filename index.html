Issue Fixed

Xray was previously failing because Hysteria2 referenced missing certificate files.

The final working certificate pair is /root/cert/server.crt and /root/cert/server.key. Hysteria2 SNI was changed to bing.com, OCSP stapling was disabled, and x-ui was restarted.

Automation Scope

  • It opens TCP 443, UDP 4443, TCP 2096, TCP 2053, and SSH.
  • It creates /root/cert/server.crt and /root/cert/server.key for Hysteria2.
  • It applies BBR and UDP buffer tuning through /etc/sysctl.d.
  • It does not publish VPN secrets; panel database backups remain separate.

Network Tuning

net.ipv4.tcp_congestion_control = bbr
net.core.default_qdisc = fq
net.core.rmem_max = 16777216
net.core.wmem_max = 16777216

Azure NSG / Firewall Requirements

  • Allow inbound TCP 443
  • Allow inbound UDP 4443
  • Allow inbound TCP 2096 for Clash subscription access
  • Optional: allow TCP 2053 only from trusted admin IPs for panel access
